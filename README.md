# Hangman Game
This Python-based Hangman game challenges players to guess words letter by letter, with visual feedback and scoring. UI elements like screens and icons were crafted in Figma for a polished look, making it both fun and educational.

Features
1. Randomized word generation and gameplay.
2. Scoring logic and user interactions.
3. Modular functions for reusability.
4. Unit testing for core features.

Technologies used
1. Language: Python
2. Design: Figma

Installation
- Clone the repository: git clone https://github.com/afreenportfolio/hangman-game-python-figma.git
- Install dependencies if needed: (Basic Python setup)

Usage
1. Run the game: python hangman.py
2. Guess letters to reveal the word.
